# 平均会心率計算ツール

注意：AIに作成していただいたコードとREADMEで試しに公開しているため実際に実行する際ご注意ください。

ゲーム内における**加重平均会心率**を算出するWebツールです。
恒常的な会心率に加え、条件・恒常それぞれの事象を複数登録することで、ゲーム全体を通じた会心率の平均値を計算します。

##　ツールの実行

https://Goya5050.github.io/average-critital-calculator/calculator.html

---

## 機能

- 恒常会心率とゲーム時間をもとにした加重平均の計算
- 事象の動的な追加・削除
- 条件発動 / 恒常発動の切り替え
- 各事象の稼働率（uptime）と平均への寄与量の内訳表示
- 最小・最大会心率の表示

---

## 使い方

### 基本設定

| 項目 | 説明 |
|------|------|
| 恒常会心率 | 装備やスキルによる常時有効な基礎会心率（%） |
| ゲーム時間 | 計測対象の区間（秒）。1戦闘・1ダンジョンなど任意で設定 |

### 事象の登録

「＋ 事象を追加」ボタンから会心率上昇事象を追加できます。

#### 条件発動

バフの発動タイミングが限られる事象（スキル使用時、被弾時など）。

| 入力項目 | 説明 |
|----------|------|
| 発動回数 | ゲーム中に発動する合計回数 |
| 合計発動時間 | バフが有効だった時間の合計（秒） |
| 会心率上昇量 | バフが有効な間の会心率上昇値（%） |

#### 恒常発動

特定の条件下で常時有効になる事象（HPが一定以上の間など）。

| 入力項目 | 説明 |
|----------|------|
| 発動継続時間 | バフが有効だった時間（秒） |
| 会心率上昇量 | 会心率上昇値（%） |

---

## 計算式

```
平均会心率 = (恒常会心率 × ゲーム時間 + Σ(各事象の上昇量 × 発動時間)) / ゲーム時間
```

各事象の「発動時間 ÷ ゲーム時間」を稼働率（uptime）として扱い、時間で重みづけした加重平均を算出します。
